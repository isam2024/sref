Summary of Features

The Midjourney Sref Tracker App is a web application designed to generate and manage sref codes for Midjourney. Key features include:

  • Generation of unique and random sref codes.
  • Local storage of saved sref sets using IndexedDB.
  • A user-friendly dashboard for managing saved sets.
  • Configurable randomness level for code generation.
  • Integration as a tab within a main dashboard UI.
  • Settings page for configuring application settings, including Ollama endpoint connection.
  • Persistent storage of settings using localStorage.
  • Connection testing for Ollama endpoint with visual feedback.
